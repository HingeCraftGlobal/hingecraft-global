### Prompt Structure
Each prompt includes:
- **prompt_id:** Unique identifier
- **sequence:** Sequential number (1-2000)
- **title:** Descriptive title
- **system_prompt:** GPT system instruction
- **user_prompt:** User-facing prompt with context
- **expected_output_length:** Target word count
- **tokens:** Estimated token usage
- **template:** Template reference
- **project/content_type/tone:** Metadata tags
