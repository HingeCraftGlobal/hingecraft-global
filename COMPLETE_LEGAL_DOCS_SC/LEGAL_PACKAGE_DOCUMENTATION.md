## USAGE INSTRUCTIONS

### For Wix Website Integration

1. **Navigate to Wix Editor**
2. **Create Legal Pages Section**
3. **Add Embed Component to each page**
4. **Paste HTML content from respective document**
5. **Set Element ID to `legalContent`**
6. **Publish changes**

### For Print Distribution

1. Open HTML file in web browser
2. Use Print function (Ctrl+P / Cmd+P)
3. Select "Save as PDF" or print directly
4. Documents are formatted for 8.5" x 11" paper
