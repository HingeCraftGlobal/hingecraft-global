## üìù Step-by-Step Instructions

### Step 1: Access SEO Settings in Wix

1. **Open Wix Editor:**
   - Log in to your Wix account
   - Go to your site editor
   - Navigate to your **homepage**

2. **Access Page Settings:**
   - Click on the page name in the top bar (or right-click on the page)
   - Select **"Page Settings"** or **"SEO"**

   **Alternative Method:**
   - Go to **Pages** in the left sidebar
   - Click on your homepage
   - Click the **"SEO"** tab

### Step 2: Set Homepage Title

1. **In the SEO Settings:**
   - Find the **"SEO Title"** or **"Page Title"** field
   - Enter the following title:

   ```
   HingeCraft Global - Charter for Abundance & Resilience
   ```

   **Best Practices:**
   - Keep it under 60 characters (this title is 58 characters - perfect!)
   - Include your main keyword: "HingeCraft Global"
   - Include value proposition: "Charter for Abundance & Resilience"
   - Make it compelling and descriptive

2. **Preview the Title:**
   - Wix will show you a preview of how it appears in search results
   - Make sure it's not truncated (if it is, shorten it)

### Step 3: Set Meta Description

1. **In the SEO Settings:**
   - Find the **"Meta Description"** field
   - Enter the following description:

   ```
   Join HingeCraft Global and support the Charter for Abundance & Resilience. Building resilient systems that protect communities and create sustainable abundance for all. Sign the charter today.
   ```

   **Best Practices:**
   - Keep it under 160 characters (this description is 156 characters - perfect!)
   - Include a call to action
   - Include key keywords
   - Make it compelling to encourage clicks

### Step 4: Set SEO-Friendly URL (Optional but Recommended)

1. **In Page Settings:**
   - Find the **"URL"** or **"Page URL"** field
   - Set it to: `/` (for homepage) or `/home`
   - Make sure it's clean and readable

### Step 5: Add Keywords (Optional)

1. **In SEO Settings:**
   - Find the **"Keywords"** field
   - Add relevant keywords (comma-separated):

   ```
   HingeCraft, Charter for Abundance, Resilience, Membership, Global Initiative, Sustainability, Community, United Nations, Policy, Abundance, Resilience Charter
   ```

   **Note:** Keywords are less important for SEO now, but can still help with internal organization.

### Step 6: Set Social Sharing Image (Recommended)

1. **In SEO Settings:**
   - Find the **"Social Sharing Image"** or **"OG Image"** field
   - Upload an image (1200x630 pixels recommended)
   - This image appears when your site is shared on social media

### Step 7: Configure Site-Wide SEO Settings

1. **Go to Site Settings:**
   - Click **Settings** in the left sidebar
   - Go to **SEO (Google)**

2. **Set Default Site Title:**
   - Enter: `HingeCraft Global - Charter for Abundance & Resilience`
   - This is used as a fallback for pages without specific titles

3. **Set Default Meta Description:**
   - Enter: `Join HingeCraft Global and support the Charter for Abundance & Resilience. Building resilient systems that protect communities and create sustainable abundance for all.`
   - This is used as a fallback for pages without specific descriptions

### Step 8: Enable SEO Features

1. **In SEO Settings:**
   - Make sure **"Enable SEO"** is turned ON
   - Enable **"Structured Data"** (helps with rich results)
   - Enable **"Sitemap"** (automatically generated by Wix)

### Step 9: Publish Your Changes

1. **Save All Changes:**
   - Click **"Save"** or **"Done"** in the SEO settings
   - Click **"Publish"** in the top right corner
   - **Important:** Changes only take effect after publishing!

## üîß Advanced SEO Settings

### Structured Data (Schema Markup)

Wix automatically adds structured data, but you can enhance it:

1. Go to **Settings** ‚Üí **SEO (Google)**
2. Enable **"Structured Data"**
3. Wix will automatically add Organization, WebSite, and other relevant schemas

### Canonical URL

1. In page SEO settings
2. Set canonical URL to your homepage URL
3. Prevents duplicate content issues

### Robots Meta Tags

1. In page SEO settings
2. Make sure **"Allow search engines to index this page"** is enabled
3. Don't use "noindex" unless you want to hide the page from search

## üêõ Troubleshooting

### Title Not Showing in Search Results

**Problem:** Your custom title isn't appearing in Google search.

**Solutions:**
1. Wait 1-2 weeks for Google to re-crawl your site
2. Request re-indexing in Google Search Console
3. Make sure you published the site after making changes
4. Check if Google is using a different title (they sometimes do this)

### Meta Description Not Showing

**Problem:** Your meta description isn't appearing in search results.

**Solutions:**
1. Google may write its own description if it thinks it's better
2. Make sure description is unique and compelling
3. Wait for Google to re-crawl
4. Request re-indexing

### Changes Not Taking Effect

**Problem:** Changes you made aren't showing up.

**Solutions:**
1. **Did you publish?** Changes only work after publishing
2. Clear browser cache
3. Wait a few minutes for changes to propagate
4. Check in incognito/private browsing mode
