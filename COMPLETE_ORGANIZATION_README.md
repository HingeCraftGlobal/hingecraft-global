# HingeCraft Global - Complete Data Organization System

## üìã Overview

This directory contains a **complete Python-based organization system** for ALL HingeCraft data, projects, and database content. Everything is segmented, organized, and documented with full project descriptions.

## üéØ Purpose

The organization system ensures that:
- ‚úÖ All HingeCraft data is segmented by project
- ‚úÖ All projects have complete descriptions
- ‚úÖ All database data is organized and accessible
- ‚úÖ Everything is available in Python format
- ‚úÖ Complete project structure is documented

## üìÅ Files

### Core Organization Files

1. **`organize_all_hingecraft_data.py`**
   - Main organization script
   - Scans all projects and files
   - Segments database data by project
   - Generates JSON and Python outputs

2. **`extract_and_organize_database_data.py`**
   - Extracts all data from database SQL files
   - Parses INSERT statements
   - Organizes data by table

3. **`HINGECRAFT_COMPLETE_ORGANIZATION.py`**
   - **MASTER organization file**
   - Contains ALL project descriptions
   - Contains ALL database table information
   - Helper functions for accessing data
   - **This is the main file to import and use**

4. **`hingecraft_organized_data.py`**
   - Auto-generated Python structure
   - Contains all projects and data
   - Generated by `organize_all_hingecraft_data.py`

5. **`hingecraft_complete_organization.json`**
   - JSON format of complete organization
   - Machine-readable format
   - Generated by `organize_all_hingecraft_data.py`

6. **`hingecraft_database_data.json`**
   - Extracted database data in JSON format
   - Generated by `extract_and_organize_database_data.py`

## üöÄ Usage

### Import the Complete Organization

```python
from HINGECRAFT_COMPLETE_ORGANIZATION import (
    ALL_HINGECRAFT_PROJECTS,
    ALL_DATABASE_DATA,
    PROJECT_DESCRIPTIONS,
    get_project_data,
    get_database_table_data,
    get_all_projects,
    get_all_database_tables,
    get_projects_by_category,
    get_database_tables_by_project
)
```

### Get Project Information

```python
# Get all projects
projects = get_all_projects()
print(f"Total projects: {len(projects)}")

# Get project data
project_data = get_project_data("database_system")
print(project_data["full_description"])

# Get projects by category
infrastructure_projects = get_projects_by_category("Infrastructure")
```

### Get Database Table Information

```python
# Get all database tables
tables = get_all_database_tables()
print(f"Total tables: {len(tables)}")

# Get table data
table_data = get_database_table_data("donations")
print(f"Records: {table_data['record_count']}")
print(f"Description: {table_data['description']}")

# Get tables for a project
payment_tables = get_database_tables_by_project("payment_system")
```

### Access All Data

```python
# Access all projects
for project_id, project_info in ALL_HINGECRAFT_PROJECTS.items():
    print(f"{project_id}: {project_info['name']}")
    print(f"  Category: {project_info['category']}")
    print(f"  Status: {project_info['status']}")
    print(f"  Completion: {project_info['completion_percentage']}%")

# Access all database tables
for table_name, table_info in ALL_DATABASE_DATA.items():
    print(f"{table_name}: {table_info['record_count']} records")
    print(f"  Description: {table_info['description']}")
```
